SQL Practice App API

This is the backend API for the SQL Practice App. To use the full application, make sure both the frontend and backend are running.

Available Endpoints:

POST /api/execute-sql - Execute SQL queries

GET /api/schema - Get database schema information

Example Usage:

// Execute a SQL query

fetch('http://localhost:5000/api/execute-sql', {

method: 'POST',

headers: { 'Content-Type': 'application/json' },

body: JSON.stringify({ query: 'SELECT * FROM users LIMIT 10' })

})

.then(response => response.json())

.then(data => console.log(data));

For a better experience, use the frontend application at http://localhost:3000
